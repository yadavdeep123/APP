# Smart Campus Navigation App

## Overview
The Smart Campus Navigation App is designed to assist new students and visitors in navigating large campuses. By providing both indoor and outdoor navigation features, the app leverages local landmarks to enhance the user experience and ensure that users can easily find their way around.

## Features
- **Indoor and Outdoor Navigation**: Users can navigate through both indoor and outdoor spaces on campus.
- **Landmark-Based Navigation**: The app uses local landmarks to guide users, making navigation intuitive.
- **User-Friendly Interface**: A clean and simple interface that allows users to easily access navigation features.
- **Search Functionality**: Users can search for specific locations, buildings, or landmarks on campus.
- **Real-Time Location Tracking**: The app provides real-time updates on the user's location to enhance navigation accuracy.

## Technology Stack
- **Flutter**: A UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.
- **Mapbox**: A mapping platform that provides customizable maps and location services.

## Project Structure
The project is organized into several directories, each serving a specific purpose:

- **lib/**: Contains the main application code, including screens, models, services, and providers.
- **assets/**: Contains static assets such as maps, fonts, and data files.
- **test/**: Contains unit and widget tests to ensure code quality and functionality.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd smart-campus-navigation
   ```

2. **Install Dependencies**:
   Ensure you have Flutter installed, then run:
   ```bash
   flutter pub get
   ```

3. **Run the Application**:
   Use the following command to run the app on an emulator or physical device:
   ```bash
   flutter run
   ```
