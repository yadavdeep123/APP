Use this exact workflow.

Collect real geospatial data
Campus boundary polygon
Building footprints
Entrances/exits
Landmarks (library, admin block, hostels, canteen, etc.)
Floor-wise indoor plans (if available in CAD/PDF)
Convert to GeoJSON
Use QGIS (best free tool) to georeference and export
Keep coordinate system as WGS84 (EPSG:4326, lat/lng)
Prepare outdoor file
Replace content in:
campus_outdoor.geojson
Include features with properties like:
name
type (building, landmark, gate, parking)
description
Prepare indoor floor files
Create one GeoJSON per floor:
building_a_floor1.geojson
same pattern for all buildings/floors

nav_node: true
id: unique node id
nav_edges: [connected_node_id_1, connected_node_id_2]
Update campus metadata

campus_info.json
landmarks.json
Wire Mapbox tokens
Public token in app constant and platform files
Then run:
flutter pub get
flutter run
